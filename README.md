## Over deze bestanden

Vertaling van Engelse liedteksten t.b.v. het programmaboekje van
L.S.K.O. Collegium Musicum, zomer 2014. Op Github gezet opdat mede-CMers
de vertaling kunnen volgen en zich ertegenaan bemoeien.
